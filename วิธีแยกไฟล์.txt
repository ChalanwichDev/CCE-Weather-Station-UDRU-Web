** วิธีแยกไฟล์ script ออกจาก HTML ***
1. สร้างไฟล์ที่โฟลเดอร์ public/js/[ชื่อไฟล์ .js]
2. ก็อปโค้ดที่อยู่ใน tag <script>...</script> ทั้งหมดไปวางในไฟล์ที่สร้าง *ไม่ต้องเอา <script>...</script>
3. ลบ tag <script>...</script> + code ใน tag ที่อยู่ในหน้า HTMLออก 
4. ให้ใช้คำสั่ง <script src="{{asset('js/[ชื่อไฟล์ .js]')}}"></script> วางแทน tag สคริปต์ ที่ลบออกในหน้า HTML
